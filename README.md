# GL Colour Grading

OpenGL based colour grading utility

## Overview

* Node based shaders
* Pipelined shader passes
* Pass intercept for intermediate tuning

## Tooling

* White balance
* Saturation
* Filmic curves and contrast
* Lift/Gamma/Gain
* LUT baking
* Sharpening masks
* Three-way correction
* Fast correction
* Curves
* Matching
* Qualifiers
* Masks
